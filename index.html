Projects

Chess Game

Chess Game

This program is a two-player chess game.


Development

This chess game was built on Eclipse IDE using Java object-oriented programming. Unit tests were written for every class to minimize bugs. The GUI was implemented using Java Swing.


Features
  • Swap turns between two players
  • Prevents illegal moves, such as:
    • Jumping over other pieces
    • Capturing another piece of the same color
    • Moving into check
  • Promotes pawn when it reaches the end of the board
  • Detects a checkmate
